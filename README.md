# ☁️ **CloudSketch – Visual Terraform Editor**

**Drag & drop Azure infrastructure → generates Terraform HCL → 1-click deploy**

---

## ✨ **Features**

**React Flow drag & drop**

- **25+ Azure nodes** : `VNet`, `AKS`, `VM`, `CosmosDB`, etc.

**Smart containers**

- `Resource Group` and `VNet` **auto-resize**

**Context menu editing**

- Right-click → edit **name**, **location**, **size**, **tags**, etc.

**Smart placement**

- **Closest parent detection** on drop

**Debug tools**

- **Hitbox debug mode**

---

## 🚀 **Quick Start**

```bash
git clone https://github.com/Dialsyp/CloudSketch.git
cd CloudSketch
npm install
npm run dev
```

Open :

```
http://localhost:3000
```
